# Customer Trend Data Analysis using SQL

## Project Overview
This project focuses on analyzing customer shopping and purchasing trends using SQL to generate meaningful business insights.

## Tools Used
- SQL (MySQL)

## Key Analysis
- Customer-wise revenue analysis
- Monthly purchase trend analysis
- Repeat vs new customer identification
- Average order value calculation

## SQL Concepts Used
- SELECT, WHERE
- GROUP BY, HAVING
- JOINS
- SUBQUERIES
- AGGREGATE FUNCTIONS

## Dataset
Publicly available retail customer shopping dataset.
